# Awesome 3D Prompts

[English](README.md) | 中文

---

一个收集高质量 3D 生成提示词（Prompts）的开源项目。

## 📖 项目简介

本项目致力于收集和整理优秀的 3D 生成提示词，帮助 3D 创作者、设计师和开发者快速找到适合的提示词，提高 3D 内容生成效率。

## 🎯 目标

- 收集高质量、实用的 3D 生成提示词
- 提供不同风格的提示词分类
- 持续更新和维护提示词库
- 为 3D 内容生成提供参考和灵感

## 📂 项目结构

```
awesome-3d-prompts/
├── README.md          # 项目说明文档（英文）
├── README_ZH.md       # 项目说明文档（中文）
├── prompts/           # 提示词分类目录
│   ├── characters/    # 角色类提示词
│   ├── objects/       # 物体类提示词
│   ├── scenes/        # 场景类提示词
│   └── styles/        # 风格类提示词
└── examples/          # 示例和演示
```

## 🔧 支持的 3D 生成工具

本项目收集的提示词适用于多种 3D 生成工具，包括但不限于：

- Stable Diffusion 3D
- Shap-E
- Point-E
- DreamFusion
- 其他 3D 生成模型

## 🤝 贡献

欢迎提交 Pull Request 来分享优秀的 3D 提示词！

请阅读我们的[贡献指南](CONTRIBUTING.md)了解行为准则和提交 Pull Request 的流程。

## 📝 提示词格式

每个提示词文件应包含以下信息：

- **标题**: 提示词的简短描述
- **提示词**: 完整的提示词文本
- **分类**: 所属类别
- **风格**: 生成风格
- **参数**: 推荐参数设置（如有）
- **示例**: 生成的示例图片或链接（如有）

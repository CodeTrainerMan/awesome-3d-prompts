# Cyberpunk Neon Style

**Category**: Artistic Style  
**Style**: Cyberpunk, Neon, Futuristic  
**Tool**: All 3D generation tools  
**Tags**: style, cyberpunk, neon, futuristic, aesthetic

## Prompt

Cyberpunk aesthetic style with neon lights, vibrant pink and cyan color palette, dark urban environment, holographic elements, rain-soaked streets, neon signs reflecting on wet surfaces, high contrast lighting, volumetric fog, glitch effects, retro-futuristic design, 80s synthwave vibes, dark purple and electric blue tones

## Parameters

- **Contrast**: High
- **Saturation**: Vibrant
- **Glow Intensity**: High
- **Color Palette**: Pink, Cyan, Purple, Blue
- **Lighting**: Neon, Dark base

## Description

This style prompt defines a cyberpunk aesthetic that can be combined with characters, objects, or scenes to create a futuristic, neon-lit atmosphere. Perfect for sci-fi projects and retro-futuristic designs.

## How to Use

Combine this style with other prompts:
- Character + Cyberpunk Style = Cyberpunk character
- Scene + Cyberpunk Style = Cyberpunk cityscape
- Object + Cyberpunk Style = Futuristic vehicle

## Variations

- "Neon Noir" - darker, more atmospheric
- "Synthwave" - 80s retro-futuristic
- "Tech Noir" - high-tech, low-life aesthetic

## Example Use Cases

- Sci-fi game environments
- Music video visuals
- Concept art
- Film backgrounds

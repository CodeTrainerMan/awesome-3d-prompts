# Fantasy Warrior Character

**Category**: Fantasy Character  
**Style**: Realistic, Fantasy, High Detail  
**Tool**: Stable Diffusion 3D, Shap-E  
**Tags**: warrior, fantasy, armor, character, humanoid

## Prompt

A highly detailed 3D fantasy warrior character, full body, wearing ornate medieval plate armor with intricate engravings, holding a glowing magical sword, heroic pose, dynamic lighting, photorealistic rendering, 4k quality, detailed face with determined expression, flowing cape, battle-worn armor with scratches and details, epic fantasy art style

## Parameters

- **Resolution**: 1024x1024 or higher
- **Quality**: High/Ultra High
- **Steps**: 50-100
- **CFG Scale**: 7-9
- **Sampler**: DPM++ 2M Karras or Euler a

## Description

This prompt generates a detailed fantasy warrior character suitable for games, animations, or 3D art projects. The armor details and magical elements create an epic fantasy aesthetic.

## Variations

- Change armor style: "steampunk armor", "samurai armor", "cyberpunk armor"
- Modify weapon: "two-handed axe", "bow and arrow", "staff"
- Adjust mood: "dark warrior", "noble paladin", "berserker warrior"

## Example Use Cases

- Game character models
- Concept art

- Animation projects
